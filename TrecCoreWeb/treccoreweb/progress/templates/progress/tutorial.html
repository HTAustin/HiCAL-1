    Tutorial

    Please read the tutorial carefully. If you have any questions, please email one of the study coordinators.

    1. Intro

    For this study, you are required to review and search for documents using our designed document-judging interfaces. You will be asked to use a judging interface to view document summaries or the full document content and judge the document's relevance to a given topic.

    You are required to complete 5 tasks. Each task consists of a pre-task questionnaire, a judging phase, and a post-task questionnaire. During the judging phase, you will be provided with a topic description (e.g. Radio Waves and Brain Cancer) that describes what documents you should look for and what is considered relevant to that topic.     You should spend a minimum of 1 hour at each task. We will collect timing information while you are completing the tasks (Section __).

    2. Pre-task questionnaire

    In the pre-task questionnaire, you will be asked few questions about the topic to better understand your familiarity with the topic and your knowledge level. You will also be given the topic description. Please read it and answer the questions provided.

    4. Judging criteria

    you should use one of the document-judging interfaces to look for document that are relevant to the topic You will be asked to mark the relevance of reviewed documents with either "highly relevant", "relevant", "not relevant". If you feel like the document has provided enough material to match with the topic description and information need, you should judge the document as "relevant". If you feel like the document is close to the topic description, but missing some key information, you should judge the document as "relevant". If you feel like the document is off topic or not what the topic description is asking for, you should judge the document as "not relevant".

    4.1 How to judge

    While viewing a document you can judge it in two ways, using keyboard shortcuts or using your mouse.

    4.1.a. Keyboard shortcuts

    To speed up the process of judging documents, you can use the following keyboard shortcuts to judge documents.

    S: marks the document as "highly relevant"

    K: marks the document as "relevant"

    H: marks the document as "not relevant"

    Clicking on one of those keys while viewing document will mark the judgment of the document.

    We will collect timing information and associated computer usage data unobtrusively during both phases of the study. This study will have you complete five phases. Each phase will have you judge one single topic and it takes around one hour. Each time you start a new phase, we will ask you some pre-phase related questions to assess your knowledge on the subject of the topic. After completing the pre-phase questions, you will be provided with an interface to review documents according to this topic. The interface will include information on the topic you need to judge, a search interface to search for the documents using our search engine, a document and summary review interface. Your task for each phase is to label documents using our provided interface. You can judge as many documents as you want. Once you believe you know the correct label for this document, you should click on the relevant/on-topic/non-relevant button and submit your label. After the phase lasts for one hour, you will be forwarded to a post-phase page to answer some follow up questions on how you felt during the process.
